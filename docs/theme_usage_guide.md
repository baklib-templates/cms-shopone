# Shop 模板使用说明文档

## 1. 模板概览 (Theme Overview)

### 模板名称

**Shop**

### 模板描述

Shop 是一款专业的电商产品知识库模板，专为构建产品帮助中心、产品手册、操作指南、视频教程和 FAQ 问答系统而设计。该模板基于 Baklib CMS 平台开发，提供了丰富的功能模块和灵活的配置选项，能够满足电商企业、产品团队和客户支持部门的各种知识管理需求。

模板采用现代化的响应式设计，支持多语言切换，提供了完整的页面类型支持，包括首页、产品列表、产品详情、视频列表、视频详情、FAQ 列表、新闻列表、文章详情、搜索页面和标签页面等。通过灵活的配置系统，用户可以轻松自定义首页版块、导航菜单、配色方案等，打造符合品牌形象的知识库网站。

### 适用场景

Shop 模板适用于以下业务场景：

1. **电商产品知识库**
   - 为电商平台提供产品使用指南、常见问题解答
   - 展示产品分类和详细信息
   - 帮助用户快速找到所需的产品帮助信息

2. **产品帮助中心**
   - 构建专业的产品帮助文档中心
   - 提供结构化的知识库导航
   - 支持多层级分类和标签系统

3. **产品使用手册**
   - 发布详细的产品操作手册
   - 提供图文并茂的使用说明
   - 支持视频教程嵌入

4. **视频教程中心**
   - 集中展示产品视频教程
   - 支持视频分类和频道管理
   - 提供视频播放和浏览功能

5. **FAQ 问答中心**
   - 构建常见问题解答系统
   - 支持多层级 FAQ 分类
   - 提供快速搜索和浏览功能

6. **企业新闻中心**
   - 发布企业新闻和公告
   - 展示最新动态和活动信息
   - 支持文章分类和标签

### 主要特性

1. **多版块首页设计**
   - 支持 FAQs 版块展示，可自定义导航菜单和描述
   - 支持 Products 版块展示，可按分类显示产品列表
   - 支持 Topics 版块展示，提供话题导航功能
   - 支持 News 版块展示，可配置内容栏目和描述
   - 支持 Contact 版块展示，提供多种联系方式（Email、Phone、Chat、Contact Form）

2. **产品管理功能**
   - 产品列表页面，支持分类筛选
   - 产品详情页面，展示完整产品信息
   - 产品分类管理，支持自定义分类名称和值
   - 产品按分类展示功能

3. **视频教程支持**
   - 视频列表页面，支持频道分类
   - 视频详情页面，支持视频播放
   - 视频分类管理

4. **FAQ 问答系统**
   - FAQ 列表页面，支持多层级分类
   - FAQ 详情页面
   - 可折叠的 FAQ 展示方式

5. **内容管理功能**
   - 新闻列表页面
   - 文章详情页面（Post）
   - 普通页面（Page）
   - 支持标签系统
   - 支持搜索功能

6. **导航和菜单系统**
   - 页头导航菜单，支持多级菜单结构
   - 页脚菜单，支持分类展示
   - 面包屑导航
   - 自定义菜单格式支持

7. **自定义配置**
   - 站点 Logo 配置
   - Request Form URL 配置
   - 自定义字体支持
   - Head HTML 代码注入
   - 页脚版权信息配置
   - 社交媒体链接配置

8. **多语言支持**
   - 支持中文简体（zh-CN）
   - 支持英文（en-US）
   - 支持其他语言扩展

9. **配色方案**
   - 提供多种预设配色方案
   - 可自定义主题颜色变量（亮色单主题）

10. **响应式设计**
    - 适配桌面端、平板和移动端
    - 使用 Tailwind CSS 构建
    - 现代化的 UI 设计

11. **SEO 和性能优化**
    - 支持自定义 Meta 标签
    - 优化的页面结构
    - 图片资源优化

12. **用户交互功能**
    - 文章反馈功能
    - 搜索功能
    - 标签筛选
    - 分页功能
